## Overview  
Questo repository contiene risorse pubbliche associate al preprint RENASCENT-Q:  
- Codice LaTeX sorgente (.tex) per il documento  
- Figure generate (PNG / PDF vettoriali)  
- PDF compilato del preprint (per download diretto)  
- Simulazioni QuTiP illustrate nel testo (codice embedded nel .tex via listings)  

Le simulazioni dimostrano:  
- Riduzione entropica retrocausale (β ≈ φ⁻² ≈ 0.382)  
- Stabilizzazione topologica via anyon braiding e MZMs  
- Concurrence → 1 e entropia von Neumann → 0 in sistemi embodied  
- Collegamenti a quantum biology (microtubuli), Orch-OR esteso, radical longevity  

Non includo script Python standalone (QuTiP) qui – sono incorporati nel LaTeX per riproducibilità contestuale.
